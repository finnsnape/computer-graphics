# Computer Graphics
## Description
Program that implements both rasterised and ray traced rendering methods.

Also features more complex techniques such as:
- Translation, rotation and lookAt
- Shadows 
- Proximity lighting 
- Angle of incidence lighting 
- Ambient lighting 
- Specular lighting 
- Mirrors

## Requirements
- `clang++`
- `sdl2`

## Running
- `make`
